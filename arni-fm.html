Arni-FM - Guia de Instalação

Este script realiza automaticamente o download, cópia, descompactação e limpeza dos arquivos do Arni-FM.

Passo 1 - Baixar o Script

wget https://seu-servidor/install-arni-fm.sh

Passo 2 - Dar Permissão

chmod +x install-arni-fm.sh

Passo 3 - Executar como Root

sudo ./install-arni-fm.sh

O Script Executa Automaticamente

  • Baixa o arquivo arni-fm.tar.gz.
  • Copia o arquivo para:

/usr/local/bin

/usr/share

  • Descompacta os arquivos nos diretórios acima.
  • Remove os arquivos compactados após a instalação.
  • Finaliza a instalação automaticamente.

Diretórios Utilizados

/usr/local/bin

/usr/share
